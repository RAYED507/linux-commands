شرح أوامر لينكس

أوامر لينكس هي أوامر تُستخدم في الطرفية (Terminal) للتحكم في نظام التشغيل، وتساعد في إدارة الملفات والمجلدات وتنفيذ الأوامر المختلفة بكفاءة.

الأوامر الأساسية

1- الأمر ls

عرض محتويات المجلد الحالي.

ls -l

2- الأمر cd

التنقل بين المجلدات.

cd Documents

3- الأمر pwd

معرفة المسار الحالي.

pwd

4- الأمر mkdir

إنشاء مجلد جديد.

mkdir project

5- الأمر rm

حذف الملفات أو المجلدات.

rm file.txt

6- الأمر cp

نسخ الملفات.

cp a.txt b.txt

7- الأمر mv

نقل أو إعادة تسمية الملفات.

mv old.txt new.txt

8- الأمر chmod

تغيير صلاحيات الملفات.

chmod 755 file.sh
